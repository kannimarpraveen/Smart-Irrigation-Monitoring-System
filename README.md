# Smart Irrigation Monitoring System

## Description
Developed an IoT-based smart irrigation system using ESP32 to monitor soil moisture, temperature, and humidity in real-time. The system enables efficient water usage through data-driven irrigation decisions.

## Features
- Real-time soil moisture monitoring
- Temperature and humidity sensing
- Wi-Fi-based data transmission
- Blynk dashboard for remote monitoring
- Threshold-based irrigation alerts

## Hardware Used
- ESP32 DevKit
- Soil Moisture Sensor
- DHT11 / DHT22 Sensor
- Relay Module (for pump control)
- Power Supply

## Working
The system continuously monitors soil moisture and environmental conditions. When moisture levels fall below a threshold, alerts are generated and irrigation can be triggered manually or automatically via the IoT dashboard.

## Applications
- Smart agriculture
- Water management systems
- Home gardening automation
